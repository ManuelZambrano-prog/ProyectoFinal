# django-ai-lab-template

## Nuevo flujo de consulta de viaje

- Se agregó una página para consultar cuánto tiempo puede tardar un envío entre ciudades en función del origen, destino, tipo de carga y peso.
- El formulario de pedidos ahora permite registrar origen, destino, tipo de carga y peso, y guarda el tiempo estimado de llegada en el pedido.
- Se añadió un nuevo enlace en la barra de navegación: `Calcular tiempo`.

## Uso

1. Ingresa a la aplicación.
2. Haz clic en `Calcular tiempo` para estimar la duración de un viaje.
3. Usa `Pedidos` -> `+ Nuevo Pedido` para registrar un pedido y obtener el tiempo estimado de llegada.
